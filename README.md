# AI Radar Pathfinding System

## Overview

This project models an aircraft navigation problem where routes must be optimized across radar-monitored terrain while minimizing cumulative detection risk.

The system combines graph theory, probabilistic modelling and heuristic search algorithms to identify optimal paths.

## Technologies

- Python
- NumPy
- NetworkX

## Algorithms

- A* Search
- Heuristic Optimization
- Graph Theory

## Features

- Radar coverage modelling
- Detection probability calculation
- Route optimization
- Terrain visualization
